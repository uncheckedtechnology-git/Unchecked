📱 App Tech Stack Guide

Everything you need to know to build a modern Android & iOS mobile application

Based on the "Unchecked" Project · 2026
1

Language & Framework

💻

JavaScript Language

Write once, run on both Android and iOS. The most popular language for cross-platform mobile apps.

⚛️

React Native Framework

Industry-standard framework by Meta. Renders real native components—not webviews—so the app feels fast and smooth.

🚀

Expo Toolchain

Simplifies React Native development. Handles native configs for you and provides free cloud builds (EAS).

2

Backend & Database (Firebase)

🔐

Firebase Auth Authentication

Secure user login and signup. Supports Email/Password, Google, Apple, and Phone logins.

🗄️

Cloud Firestore Database

Real-time NoSQL database. Stores user profiles, matches, chat messages, and swipe history in the cloud.

🖼️

Firebase Storage File Storage

Stores user-uploaded photos and media files. Delivers them instantly via Google's global CDN.

3

Navigation

🧭

React Navigation Routing

Handles screen transitions with native-feeling animations. Includes Stack Navigator, Bottom Tabs, and more.

4

UI & Animations

✨

Reanimated Animations

Silky-smooth 60fps animations (card swiping, transitions) running directly on the native UI thread.

👆

Gesture Handler Touch

Captures swipes, pinches, and long-presses—essential for Tinder-style interactions.

🎨

Skia + SVG Graphics

High-performance 2D rendering for custom charts, gradients, and advanced visual effects.

5

Device Features

📸

Image Picker Camera

Access the phone's camera or photo gallery to let users upload profile pictures.

📍

Location GPS

Get the user's real-time GPS location for proximity features like "people near you."

📳

Haptics Vibration

Subtle vibrations on button taps, matches, and actions—adds a premium, tactile feel.

6

Build & Deploy

📦

EAS Build Distribution

Expo's free cloud service to compile your code into installable .apk (Android) and .ipa (iOS) files.

🧪

Jest Testing

Automated testing to make sure new code changes don't break existing features before release.


✓

Quick Checklist for a New App

  • Framework: React Native + Expo
  • Backend: Firebase (Auth + Firestore + Storage)
  • Navigation: React Navigation
  • Animations: Reanimated + Gesture Handler
  • Hardware: Expo APIs (Camera, Location, Haptics)
  • Distribution: EAS Build (Free Cloud APK/IPA)
